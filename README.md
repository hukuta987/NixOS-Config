# NixOS Config

## May be broken!

Minimalistic dwm config for NixOS. If you really want to use it, I'd prefer editing some lines in **configuration.nix**, atleast nvidia drivers.

# Installation
```

sudo curl -L https://raw.githubusercontent.com/hukuta987/NixOS-Config/refs/heads/main/configuration.nix -o /etc/nixos/configuration.nix

sudo nixos-rebuild switch
```
